Azure NetApp Files supports fine-grained access control for volumes using export policies, Unix permissions, access control lists (ACLs), and SMB share ACLs. You can also configure network features to optimize connectivity for your volumes.

In this module, you learn how to manage access to NFS, SMB, and dual-protocol volumes in Azure NetApp Files. You learn to configure export policies to control who can mount a volume and what level of access they have. You also learn to set Unix permissions, manage ACLs for NFSv4.1 volumes, and manage SMB share ACLs using Windows tools.

Finally, you learn how to configure network features and Virtual WAN for Azure NetApp Files volumes.
