The goal of this module was to help you manage access to a volume in Azure NetApp Files.

You learnt how to manage volume access by:

- Configuring export policies and Unix permissions for NFS and dual-protocol volumes
- Managing access control lists (ACLs) for NFS and SBM volumes
- Configuring Virtual WAN for Azure NetApp Files

You also performed hands-on exercises to configure an export policy for volume, configure Unix permissions and change ownership mode, configure network features for Azure NetApp Files volume.
