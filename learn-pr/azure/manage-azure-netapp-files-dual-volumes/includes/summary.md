The goal of this module was to help you create and manage dual-protocol volumes in Azure NetApp Files.

You learned how to manage a dual-protocol volume by:

- Creating Azure NetApp Files dual-protocol volumes
- Allowing local NFS users with LDAP to access a dual-protocol volume

You also performed hands-on exercises to create a dual-protocol volume for Azure NetApp Files.
