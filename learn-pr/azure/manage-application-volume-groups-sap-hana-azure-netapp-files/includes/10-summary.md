The goal of this module was to help you manage application volume groups for SAP HANA in Azure NetApp Files, including deploying a SAP HANA host, adding volumes to a SAP HANA system, and managing volumes in the application volume group.

You learned how to manage application volume groups for SAP HANA by:

- Deploying an SAP HANA host using application volume groups
- Adding volumes for an SAP HANA system in a multiple-host setup
- Configuring HANA System Replication (HSR) for high availability
- Using cross-region replication for disaster recovery
- Managing volumes in an application volume group for SAP HANA

You also performed hands-on exercises to deploy the first SAP HANA host, add hosts to a multiple-host SAP HANA system, and manage volumes in an application volume group for SAP HANA.
