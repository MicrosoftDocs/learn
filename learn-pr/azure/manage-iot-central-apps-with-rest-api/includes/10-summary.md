Your company plans to deploy IoT devices in stores to enable remote environmental monitoring and control using IoT Central. You were asked to integrate device management and control functions into the existing store management application so that store managers can continue to use a single application.

The IoT Central REST API lets you manage your IoT Central application and devices programmatically. You created an API token to authorize access to your application, and used the REST API to manage device templates and devices. You used the REST API to control the devices in your stores by setting properties and sending commands. You also programmatically configured your IoT Central application to continuously export data to Azure blob storage where the data can be further analyzed.

Integrating the IoT Central application functionality into your existing store management application makes it much easier for operators and store managers to analyze and respond to data from your stores because they have everything they need in a single application.
