The goal of this module was to help you create and manage SMB volumes in Azure NetApp Files.

You learned how to manage an SMB volume by:

- Creating an SMB volume
- Controlling access to an SMB volume through permissions
- Using DFS Namespaces with Azure NetApp Files

You also performed hands-on exercises to create an SMB volume for Azure NetApp Files and enable Continuous Availability feature in Azure NetApp files SMB volumes.
