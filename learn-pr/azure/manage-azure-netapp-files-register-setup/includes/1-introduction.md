# Register and Set Up Azure NetApp Files

Learn to register and setup Azure NetApp Files by registering the NetApp Resource Provider and create a NetApp account. You will also learn how to delegate a subnet to Azure NetApp Files and request access to a region.

## Learning Objectives

By completing this module, you’ll be able to:

* Register to the Azure Resource Provider
* Create a NetApp account
* Delegate a subnet to Azure NetApp Files

## Prerequisites

* Azure NetApp Files Security
