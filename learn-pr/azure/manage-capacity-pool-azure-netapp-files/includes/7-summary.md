The goal of this module was to help you create and manage capacity pools for Azure NetApp Files.

You learned how to manage a capacity pool by:

- Setting up a new manual QoS capacity pool
- Changing a capacity pool to use manual QoS
- Monitoring the throughput of a manual QoS capacity pool
- Modifying the allotted throughput of a manual QoS volume

You also performed hands-on exercises to create a capacity pool in Azure NetApp Files, manage a manual QoS capacity pool, and manage billing in Azure NetApp Files.
