The goal in this module was to help you create and manage Active Directory connections for Azure NetApp Files.

You also learned how to

- Modify an Active Directory connection
- Configure Active Directory DS LDAP over TLS for Azure NetApp Files
- Export root CA certification
- Enable LDAP over TLS
- Disable LDAP over TLS
- Join a Linux VM to an Active Directory domain

You performed hands-on exercises to create, modify, and manage an Active Directory connection. Also configured and Active Directory for LDAP authentication for NFS volumes.
