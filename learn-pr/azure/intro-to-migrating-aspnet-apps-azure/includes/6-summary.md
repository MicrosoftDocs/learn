Hosting ASP.NET applications in private or shared datacenters typically results in an organization incurring capital expenditures and ongoing maintenance costs for hardware and software. Migrating the application and its associated data to Azure can provide savings to the organization by offloading infrastructure costs and automating many aspects of the application maintenance requirements. Azure App Service and Azure SQL Database, when combined, provide availability, scale, security, and infrastructure management for your ASP.NET application.

In this module, you've learned what a migration process is and the various tools and options that help you decide if migration is the right choice. You've also learned how to accomplish the migration process in an efficient manner by using migration assistants, and how integration with a DevOps process can help automate application updates and deployments.
