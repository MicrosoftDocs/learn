The goal in this module was to help you create and manage NFS volumes for Azure NetApp Files.

You learned how to manage an NFS volume by:

- Creating Azure NetApp Files NFS volumes
- Configuring NFSv4.1 ID domain and Kerberos
- Enabling Active Directory Domain Services and LDAP authentication
- Converting NFS volumes between NFSv3 and NFSv4.1

You also performed hands-on exercises to create an NFS volume in Azure NetApp Files, configure NFSv4.1 ID domain on Azure NetApp Files, and convert an NFS volume between NFSv3 and NFSv4.1.
