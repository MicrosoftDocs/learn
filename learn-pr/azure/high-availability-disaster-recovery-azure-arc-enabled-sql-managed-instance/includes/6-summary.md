In this module, you have learned the different High Availability options for Azure Arc-enabled SQL Managed Instance tiers and how these options work to provide and meet your High Availability requirements. You have learned how to implement Disaster Recovery for your Azure Arc-enabled SQL Managed Instance and how Point-in-time Recovery fits into your environment.

## Learning objectives

In this module, you have:

- Reviewed the different High Availability Options
- Reviewed the Disaster Recovery Options
- Configured readable secondaries and used Point-in-time Restores
