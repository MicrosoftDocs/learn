The goal of this module was to help you manage various data protection options in Azure NetApp Files.

You learned how to manage volume snapshots, volume backups, replication, availability zone, and cross-region replication by:

- Managing snapshots and backups in Azure NetApp Files
- Managing cross-region replication and disaster recovery options
- Managing availability zone volume placement for Azure NetApp Files

You also performed hands-on exercises to manage volume snapshots, volume backups, and replication for Azure NetApp Files.
