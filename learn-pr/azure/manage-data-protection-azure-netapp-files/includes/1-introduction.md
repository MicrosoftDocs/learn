Learn various ways to manage volume snapshots, volume backups, replication, availability zone, and cross region replication with Azure NetApp Files.

### Learning objectives

By completing this module, you'll be able to:

- Manage snapshots and backups in Azure NetApp Files
- Manage cross-region replication and disaster recovery options
- Manage availability zone volume placement for Azure NetApp Files

### Prerequisites

- Manage application volume groups for Oracle in Azure NetApp Files
