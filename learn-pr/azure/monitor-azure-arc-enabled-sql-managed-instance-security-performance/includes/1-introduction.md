At Tailwind Traders, in your role of managing and administering the SQL Server environments, it is imperative the environments are secure and performing optimally.  

![Image of Tailwind Traders](../media/tailwind-traders-logo.png)

In this module, you will learn specific methods of Authenticating to an Azure Arc-enabled SQL Managed Instance to meet your security requirements. You will learn what tools are available to assist in the monitoring and optimization of an Azure Arc-enabled SQL Managed Instance.

## Learning objectives

At the end of this module, you will be able to:

- Secure an Azure Arc-enabled SQL Managed Instance
- Monitor the performance of an Azure Arc-Enabled SQL Managed Instance
