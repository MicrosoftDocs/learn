You can mount an SMB volume for Windows virtual machines (VMs).

### Mount SMB volumes on a Windows client

1. Select the Volumes menu and then the SMB volume you want to mount.
2. To mount the SMB volume using a Windows client, select **Mount instructions** from the selected volume.
3. Follow the steps shown in Azure portal to mount the volume.
