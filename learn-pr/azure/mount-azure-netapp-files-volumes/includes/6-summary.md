The goal of this module was to help you mount Azure NetApp Files NFS and SMB volumes on Linux and Windows systems.

You learned how to:

- Mount NFS volumes on Linux and Windows clients
- Mount SMB volumes on Windows clients

You also performed hands-on exercises to mount NFS volumes on Linux and Windows clients, and mount SMB volumes for Windows virtual machines.

- [Mount SMB volumes for Windows virtual machines](https://learn.microsoft.com/azure/azure-netapp-files/mount-volumes-vms-smb)
