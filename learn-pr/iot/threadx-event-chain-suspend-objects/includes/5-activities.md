## Introduction

The following image contains a description of the activities for this project. The Urgent thread has priority 5, and the Routine thread has priority 15. We need two timers to send messages to the queues at periodic intervals. The Queue_1 timer sends messages only to Queue_1, and the Queue_2 timer sends messages only to Queue_2.
