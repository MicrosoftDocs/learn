In this module you learned how to manage IIS on Windows Server using IIS Manager, the Configuration Editor, and remote administration via WMSVC with delegated access through IIS Manager Users and Permissions. You used the WebAdministration PowerShell module and AppCmd.exe to script site, application pool, and configuration management, including backup and restore. You monitored worker processes, analyzed IIS logs with PowerShell, and established routine health checks. You also learned to prefer targeted application pool recycles over `iisreset` to minimize disruption.
