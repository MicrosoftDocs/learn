Contoso Camping Store would like to launch a new chatbot to enhance their customer support. The proposed chatbot should be designed to provide instant, accurate information about the store's wide range of camping products. Your goal is to create a chatbot in [Azure AI Foundry](https://ai.azure.com) that can generate responses about Contoso Camping Store product features and provide recommendations based on customer preferences. As you build it, you also apply layered mitigations so the chatbot stays grounded, relevant, and safe.

## Learning objectives

By completing this module, you're able to:

- Choose and deploy a model from the model catalog
- Upload data and create a vector index
- Set up and refine a system message
- Create and apply a content filter
- Run a manual evaluation
- Run and compare automated evaluations

## Prerequisites

- An Azure subscription – [Create one for free](https://azure.microsoft.com/pricing/purchase-options/azure-account?cid=msft_learn)
- Permission to create or manage resources in the subscription, or access to an existing hub-based project
- Familiarity with Azure and the Azure portal
