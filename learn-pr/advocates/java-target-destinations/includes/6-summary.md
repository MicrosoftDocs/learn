In this module, you learned about the main deployment types you can use for Java applications on Azure. You reviewed managed platform services, serverless options, managed Kubernetes, and virtual machines (VMs). You also learned how to choose the Azure service that best addresses your team's needs: Azure Container Apps, Azure App Service, Azure Functions, Azure Kubernetes Service (AKS), or Azure Virtual Machines.
