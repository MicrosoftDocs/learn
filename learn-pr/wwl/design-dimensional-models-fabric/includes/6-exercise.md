A retail organization needs to analyze sales performance across stores, products, customers, and time periods. In this exercise, you design and implement a star schema dimensional model in a Fabric Warehouse: creating fact and dimension tables, loading sample data, running analytical queries, and implementing slowly changing dimension (SCD) patterns.

> [!NOTE]
> You need access to a Microsoft Fabric-enabled workspace to complete this exercise. For more information, see [**Getting started with Fabric**](/fabric/get-started/fabric-trial) to enable a Fabric trial license.

This lab takes approximately **30** minutes to complete.

Launch the exercise and follow the instructions.
