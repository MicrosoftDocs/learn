In this exercise, you create a Dataflow Gen2 to connect to sample data, apply Power Query transformations to filter, clean, and shape the data, and load the results to a lakehouse table. You perform the following tasks:

- Create a Dataflow Gen2 and connect to a data source
- Apply transformations and custom columns
- Configure a lakehouse as the data destination
- Publish the dataflow and verify results

> [!NOTE]
> You need access to a Fabric-enabled workspace to complete this exercise. For information about a trial license, see [Getting started with Fabric](/fabric/get-started/fabric-trial).

This lab takes approximately **30** minutes to complete.

Launch the exercise and follow the instructions.
