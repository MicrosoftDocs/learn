In this module, you explored how Dataflows Gen2 use Power Query to connect to data sources, apply transformations, and load results to supported destinations. You saw how the Power Query Online editor records each transformation as an applied step, creating an auditable and repeatable process for data preparation. You also learned how to optimize dataflow performance using query folding, preview-only steps, and the Modern Query Evaluator. Copilot for Dataflow Gen2 can also generate and explain transformation logic with natural language.

For teams with Power Query experience, Dataflows Gen2 bring those familiar skills into enterprise-scale cloud data preparation without requiring Spark or T-SQL code. The clean, well-structured data you produce supports downstream analysts and AI experiences like Copilot and data agents across the platform.
