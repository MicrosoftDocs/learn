In this exercise, you validate a semantic model using SemPy in a Fabric notebook, then create a deployment pipeline to promote validated content from a development workspace to production. These tasks practice the **Validate → Deploy** stages of the semantic model lifecycle.

> [!NOTE]
> You need access to a Fabric-enabled workspace to complete this exercise. For information about a trial license, see [Getting started with Fabric](/fabric/get-started/fabric-trial).

This lab takes approximately **30** minutes to complete.

Launch the exercise and follow the instructions.
