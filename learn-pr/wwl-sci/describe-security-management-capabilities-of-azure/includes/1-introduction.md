
As more companies move their assets and resources into the cloud, keeping them safe is a primary consideration for all IT and security departments. Cybercrime is a multi-billion-dollar business. Failure to protect your organization can be costly because of loss of data and reputation. The growing adoption of generative AI introduces an additional category of workloads that organizations need to protect—making a unified, comprehensive approach to cloud security more important than ever.

Microsoft Defender for Cloud is a cloud-native application protection platform (CNAPP) that helps organizations protect cloud workloads, manage their security posture, embed security into development pipelines, and secure their AI applications across Azure, other clouds, and on-premises environments.

In this module, you learn about Microsoft Defender for Cloud and how it uses security policies, standards, and recommendations to improve cloud security posture. You learn how each of the three pillars of Microsoft Defender for Cloud protects against cyber threats and vulnerabilities, and you explore how Defender for Cloud extends its capabilities to protect generative AI workloads.

After completing this module, you'll be able to:

- Describe Microsoft Defender for Cloud, its core pillars, and its AI security capabilities.
- Describe how security policies, standards, and recommendations improve cloud security posture.
- Describe cloud security posture management (CSPM) and the features available through Defender CSPM.
- Describe the enhanced security features provided through Microsoft Defender plans and cloud workload protection.
- Describe DevOps security management capabilities in Microsoft Defender for Cloud.
