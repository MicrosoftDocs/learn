
You learned about Microsoft Defender for Cloud and how it uses security policies, standards, and recommendations to improve cloud security posture. You also learned how each of the three pillars of Microsoft Defender for Cloud—DevSecOps, CSPM, and CWPP—help organizations protect against cyber threats and vulnerabilities, and how Defender for Cloud extends its capabilities to protect generative AI workloads through AI security posture management and AI threat protection.

Now that you've completed this module, you should be able to:

- Describe Microsoft Defender for Cloud, its core pillars, and its AI security capabilities.
- Describe how security policies, standards, and recommendations improve cloud security posture.
- Describe cloud security posture management (CSPM) and the features available through Defender CSPM.
- Describe the enhanced security features provided through Microsoft Defender plans and cloud workload protection.
- Describe DevOps security management capabilities in Microsoft Defender for Cloud.
