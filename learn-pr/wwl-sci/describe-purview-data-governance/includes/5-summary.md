
This module introduced you to key principles of data governance and the different roles in data governance.

You learned about the two primary data governance solutions in Microsoft Purview. Data Map provides the foundation by scanning and capturing metadata across hybrid, on-premises, and multicloud environments. Unified Catalog builds on that metadata to provide tools for curation, governance, access management, data quality, and health management.

Now that you've completed this module, you should be able to:

- Describe the concepts and benefits of data governance.
- Describe what Microsoft Purview Data Map does and why it matters.
- Describe key features of Microsoft Purview Unified Catalog.
