In this exercise, you use an interactive guide to explore the Threat Intelligence Briefing Agent and create an agent in the Microsoft Security Copilot standalone experience.

## Exercise overview

The interactive guide walks you through a scenario at Datum Corporation, a global manufacturing company that deployed Microsoft Security Copilot to enhance its security operations. You're assigned as a Copilot Contributor with the security operator role. Your goal is to explore how agents work in the standalone experience and create one.

The interactive guide covers two tasks:

- **Task 1: Explore the Threat Intelligence Briefing Agent** - You review the agent's details, activity, and reports. You view the threat intelligence briefing report generated by the agent, including the input parameters and results. You also explore options to download, copy, and provide feedback on the report, view the activity map, and run the agent on demand.
- **Task 2: Create an agent in the standalone experience** - You review the three methods for creating an agent (natural language, start from scratch, and YAML upload). You then create an agent using natural language (NL2Agent) by describing the agent's purpose, reviewing the generated configuration, configuring automation and permissions settings, viewing the YAML manifest, publishing the agent, and testing it.

This interactive guide should take approximately **10** minutes to complete.
