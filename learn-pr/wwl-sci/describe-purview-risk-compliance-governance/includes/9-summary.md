
In this module, you learned how Microsoft Purview data compliance solutions help you manage and monitor your data, minimize compliance risks, and meet regulatory requirements.

The module covered auditing solutions in Microsoft Purview, including Audit (Standard) and Audit (Premium), which help organizations respond to security events, forensic investigations, and compliance obligations. You also learned about eDiscovery, a process used to identify and deliver electronic information as evidence in legal cases, and the workflow and features that Microsoft Purview eDiscovery provides to support investigations. The module then described the Microsoft Purview portal and Microsoft Purview Compliance Manager—a solution for assessing and managing compliance across multicloud environments—and explained how the compliance score helps organizations understand and improve their compliance posture. You explored Communication Compliance, an insider risk solution for detecting inappropriate messages, and learned about the broader insider risk management capabilities in Microsoft Purview. Finally, the module covered Data Lifecycle Management, which provides tools for retaining and deleting content through retention policies and labels, and Microsoft Purview Records Management, which supports the full lifecycle of regulatory, legal, and business-critical records.

Now that you've completed this module, you're able to:

- Describe audit solutions in Microsoft Purview.
- Describe eDiscovery in Microsoft Purview.
- Describe the Microsoft Purview portal and Compliance Manager.
- Describe Microsoft Purview Communication Compliance and insider risk management.
- Describe Microsoft Purview Data Lifecycle Management.
- Describe Microsoft Purview Records Management.
