
Microsoft Purview is a comprehensive set of solutions, accessed through the Microsoft Purview portal, that helps organizations govern, protect, and manage data across their entire data estate. These solutions include compliance tools that help organizations manage regulatory obligations, reduce risk, and respond to legal and security events.

In this module, you learn about Microsoft Purview data compliance solutions and how they can help your organization manage and monitor its data, minimize compliance risks, and meet regulatory requirements.

After completing this module, you'll be able to:

- Describe the Microsoft Purview portal and Compliance Manager.
- Describe Microsoft Purview Communication Compliance.
- Describe eDiscovery in Microsoft Purview.
- Describe audit solutions in Microsoft Purview.
- Describe Microsoft Purview Data Lifecycle Management.
- Describe Microsoft Purview Records Management.
