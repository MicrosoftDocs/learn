### Improved Windows Search

Improved Windows Search helps users locate files, images, and documents using natural language descriptions instead of exact file names or folder locations.

For example, users can search using phrases such as:

- "Presentation about quarterly planning"
- "Document with sustainability charts"
- "Budget spreadsheet from last month"

This experience can help users quickly locate content across their device so they can spend less time searching for information and more time focusing on work.

### Click to Do

Click to Do helps users take quick actions on text and images directly from their screen.

Examples include:

- Opening selected text in a new document
- Scheduling meetings from highlighted email addresses
- Copying or editing images
- Performing visual searches

This experience helps streamline common workflows without needing to switch between multiple applications or menus.

### Recall (preview)

Recall (preview) helps users return to previously viewed content and activities using natural language descriptions and visual context. Unlike Improved Windows Search, which focuses on locating files and content, Recall (preview) is designed to help users reconnect with work they previously interacted with throughout the day.

For example, users might search for:

- "Presentation with green charts"
- "Notes from the planning meeting"
- "Document about hiring goals"

Recall (preview) can help users return to previously viewed content and continue working more efficiently.

## Explore communication and collaboration experiences

Other Copilot+ PC experiences focus on communication, accessibility, and collaboration across meetings and conversations.

### Live Captions with translation

Live Caption with translation provides real-time captions and translations during live or recorded audio experiences.

This experience can help users:

- Follow multilingual meetings more easily
- Improve accessibility during presentations or calls
- Understand spoken content across supported languages

### Windows Studio Effects

Windows Studio Effects helps improve video and meeting experiences through AI-powered camera and audio enhancements.

Examples include:

**Camera enhancements**

- Background blur
- Automatic framing
- Eye contact adjustments

**Audio enhancements**

- Noise reduction

These features can help users maintain clearer and more focused communication during virtual meetings and presentations.

This can be especially useful during multilingual meetings, collaborative discussions, and accessibility-focused communication scenarios.
