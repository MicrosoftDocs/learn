# How Copilot+ PCs improve everyday workflows

AI capabilities in Copilot+ PCs are integrated into everyday workflows to reduce manual steps, surface relevant information, and help users move from insight to action more quickly.

These on-device AI experiences are especially valuable in situations where users need responsive performance, offline functionality, or local processing for privacy-sensitive work.

Building on the foundation of on-device and cloud-based AI introduced earlier in this module, the next step is understanding how these capabilities come together in real-world work scenarios.

## How AI supports common work tasks

Rather than requiring users to learn entirely new tools or workflows, AI experiences in Copilot+ PCs are integrated directly into Windows 11 and Microsoft 365 experiences.

Key areas where AI supports everyday work include:

- **Finding information**—AI-enhanced search and contextual suggestions help users locate files, settings, and content without relying on exact file names or folder structures.
- **Creating and refining content**—AI-assisted tools help users draft, summarize, and improve written content more efficiently.
- **Communicating across teams**—Features like Live Captions and Windows Studio Effects support clearer and more inclusive communication during meetings and collaboration.
- **Moving from information to action**—Features like Click to Do identify actionable content on screen and suggest relevant next steps.

These experiences help reduce context switching and minimize the number of manual steps required to move from information to action.

Some AI experiences may vary depending on device hardware, language, or regional availability.

## Scenario: Supporting a business expansion initiative

To understand how these capabilities work together in practice, consider the following scenario.

A team at a growing organization is planning an expansion into a new market. The project requires research, cross-functional collaboration, and communication with senior stakeholders. A project lead is responsible for coordinating the effort across marketing, finance, and sales teams.

Here’s how AI capabilities in a Copilot+ PC support the project lead through three phases of work:

### Phase 1: Research and content creation

The project lead needs to build a presentation for senior leadership. This requires locating relevant market data, competitive analysis, and internal planning documents.

Using Improved Windows Search, the project lead locates files and previous content without needing to remember exact file names or folder locations. AI-enhanced search understands context, so a search like “market demographics report from last quarter” returns relevant results even if the file name doesn’t contain those exact words.

While reviewing a report on screen, Click to Do identifies key data points and suggests actions such as copying text, opening related documents, or starting a summary. This helps the project lead move quickly from research into content creation without constantly switching applications.

To help organize and draft the presentation, the project lead uses Microsoft 365 Copilot to generate an initial outline based on the gathered research and organizational data.

### Phase 2: Collaboration

The project involves regional teams in multiple countries. To ensure all perspectives are represented, the project lead invites colleagues to present in their preferred language during a virtual meeting.

Live Captions provides real-time, on-device translation so attendees can follow the discussion more easily across languages. Captions are also available in the meeting recording for employees who couldn't attend live.

Both the project lead and team members use Windows Studio Effects to improve the meeting experience:

- **Portrait Lighting** helps speakers remain clearly visible on camera
- **Voice Focus** helps reduce background noise during conversations

These features run locally on the NPU, helping provide consistent experiences without interrupting other workflows.

### Phase 3: Delivery and follow-up

After the presentation, the project lead needs to communicate decisions and action items to stakeholders.

Instead of manually reviewing notes and recordings, the project lead uses Microsoft 365 Copilot to generate a summary of the meeting, including key decisions and assigned action items.

Using this summary, the project lead drafts a stakeholder email in Microsoft 365 Copilot and refines the tone and structure for an executive audience.

## How AI capabilities work together in this scenario

On-device AI experiences in Windows can operate in different ways depending on the feature and hardware capabilities available.

Some experiences are accelerated by the NPU on Copilot+ PCs, allowing AI models to run locally on the device for responsive, real-time experiences. Other features run locally in Windows without requiring dedicated NPU acceleration.

The table below highlights where each capability runs and how it contributes to the scenario.

| Capability | Where it runs | Role in the scenario |
|---|---|---|
| Improved Windows Search | On-device (NPU-accelerated on Copilot+ PCs) | Located relevant files using contextual search |
| Click to Do | On-device (NPU; some actions may use cloud services) | Suggested actions from on-screen content |
| Live Captions | On-device (NPU for real-time translation) | Provided real-time transcription and translation during collaboration |
| Windows Studio Effects | On-device (NPU) | Enhanced video and audio quality during meetings |
| Microsoft 365 Copilot | Cloud | Generated presentation outline and meeting recap |
| Microsoft 365 Copilot email drafting | Cloud | Drafted stakeholder communication |

> [!IMPORTANT]
> Copilot+ PC features and Microsoft 365 Copilot are complementary but separate. Copilot+ PC experiences are built into Windows and powered by the NPU. Microsoft 365 Copilot requires an eligible Microsoft 365 license.

This combination allows users to work efficiently across research, collaboration, and delivery workflows while balancing local AI responsiveness with cloud-connected AI capabilities.
