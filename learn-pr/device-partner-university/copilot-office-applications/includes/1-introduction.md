With Microsoft 365, users can simplify their daily tasks and projects using applications that they use every day. All Microsoft 365 applications—like Word, Excel, and PowerPoint—now feature Copilot built into the app experience. Copilot helps to turn data into insights, refine ideas, and summarize text—helping users optimize their workflow and prioritize the things that matter.

In this module, you'll discover how Copilot in Microsoft 365 applications help users in the context of their work within those applications. You'll also explore scenarios and steps for utilizing Copilot features throughout Microsoft 365 to better understand how this tool can help you be more efficient and manage tasks with reduced friction.

## Learning objectives

By the end of this module, you'll be able to:

- Learn how to access Copilot in each Microsoft 365 app and use it to draft, summarize, rewrite, organize, and analyze your work.
- Identify how Copilot can help you complete everyday tasks—like generating documents, cleaning up emails, building presentations, and organizing notes—faster and with higher quality.
- Save time and create better content using Copilot directly inside Word, Excel, PowerPoint, Outlook, and OneNote.
