Suppose you’re responsible for managing IT at a mid-sized organization. Windows 10 has reached its end of support, and you need to decide how to keep your organization’s devices secure, compliant, and productive. You’re evaluating three upgrade paths—Windows 11 on new devices, Windows 10 Extended Security Updates (ESU), and Windows 365 Cloud PC—to address different business needs, technical requirements, and budget constraints.

This module helps you compare these options, apply decision criteria, and choose the best Windows upgrade path for your organization’s unique situation.

## Learning objectives

By the end of this module, you'll be able to:

- Identify the core capabilities and use cases for Windows 11 on new devices, Windows 10 Extended Security Updates (ESU), and Windows 365 Cloud PC.
- Analyze organizational scenarios to determine the most appropriate upgrade or mitigation path.
- Apply decision criteria such as security, cost, hardware readiness, and cloud adoption.
- Recommend the best path forward based on each organization’s technical and business requirements.
