The software you run is fundamental to modern work, and where it comes from has real consequences. Windows might look the same on the surface—but whether it's genuine or not makes a meaningful difference in trust, security, and reliability over time.

Non-genuine software often enters a digital ecosystem quietly: a reused license, a preinstalled copy on refurbished hardware, or a download that seems to work "well enough." At first glance, it can be hard to tell the difference. Over time, that difference manifests in how dependable the device is to use, the protection it provides, and the risks you might not see until something goes wrong.

## Learning objectives

After completing this module, you'll be able to:

- Define what genuine Windows means in terms of software authenticity.
- Explain how authenticity and system integrity contribute to trust, security, and reliability.
- Distinguish between scenarios that reflect genuine and nongenuine Windows and the associated risks.
