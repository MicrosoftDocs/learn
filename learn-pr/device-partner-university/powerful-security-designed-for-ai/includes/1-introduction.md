In today's AI-powered workplace, security is foundational to how organizations protect data, identities, and devices.

As AI becomes integrated into everyday workflows, organizations need platforms that secure information by design across hardware, software, and AI experiences. In Copilot+ PCs, security is built into the platform rather than added as a separate layer.

This module explores how Copilot+ PCs combine Windows 11 Pro security, hardware-backed protections, and on-device AI processing to help organizations adopt AI confidently.

## Learning objectives

By the end of this module, you'll be able to:

- Describe how Copilot+ PCs integrate hardware-backed security with on-device AI processing  
- Explain how Windows 11 Pro security features protect data, identities, and devices  
- Summarize how Microsoft Pluton and Windows Hello Enhanced Sign-in Security protect credentials and device integrity  
- Describe how on-device AI experiences support privacy and data protection  
- Identify how IT administrators can govern on-device AI experiences using existing management tools and policies
