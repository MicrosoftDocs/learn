Before diving into specific features, it's helpful to understand how security is implemented across Copilot+ PCs.

Security is integrated across multiple layers—from the silicon to the operating system and the AI experiences that run on the device.

## Built for on-device AI

Copilot+ PCs support AI-powered workloads using a dedicated Neural Processing Unit (NPU). These processors can deliver over 40 trillion operations per second (TOPS), enabling AI workloads to run locally on the device.

The NPU helps distribute AI workloads efficiently across the system, improving responsiveness while maintaining secure, on-device processing.

This on-device AI processing model provides two primary benefits:

- **Performance**: Real-time responsiveness and lower latency for AI-powered tasks without relying on cloud processing  
- **Privacy**: Sensitive business data stays on the device, reducing exposure during AI processing  

This approach allows AI processing to occur locally while helping reduce exposure of sensitive data.

## Security and control at scale

Windows 11 Pro provides a strong security foundation across devices, identities, and data.

Copilot+ PCs extend this foundation by combining AI hardware with existing Windows security and management capabilities.

With these capabilities, organizations can:

- Run AI workloads locally using NPUs with 40+ TOPS  
- Protect credentials and sensitive data with Microsoft Pluton and Secured-core PC capabilities  
- Enable on-device AI experiences like Recall (preview) and Click to Do  
- Apply policies and controls to manage how AI features are enabled and used  

These capabilities help organizations adopt AI while maintaining visibility, control, and compliance.

## Security designed for the AI workplace

Copilot+ PCs integrate security across hardware, software, and AI experiences. These capabilities can be understood across three key areas:

| Area | What it includes | Why it matters |
|------|-----------------|----------------|
| **Hardware security** | Microsoft Pluton, Secured-core PC protection, TPM 2.0 | Establishes a hardware root of trust and protects credentials and system integrity |
| **On-device AI with privacy built in** | NPU-powered experiences like Recall and Click to Do | Keeps AI processing local, helping reduce data exposure and protect sensitive information |
| **Enterprise governance and compliance** | Microsoft Intune, policy controls, device and identity management | Enables IT administrators to manage, control, and enforce security policies at scale |

> [!NOTE]
> Copilot+ PCs combine local AI processing with hardware-backed security and centralized management. This allows organizations to apply consistent protection across both device-level and AI-driven experiences.

The following sections explain each of these areas in more detail.

## Hardware security in Copilot+ PCs

Every Copilot+ PC is a **Secured-core PC**, with protections across hardware, firmware, and the operating system.

## Secured-core protection

Key technologies include:

| Component | What it does | Security benefit |
|-----------|--------------|------------------|
| **Microsoft Pluton** | Integrated security processor in the CPU | Protects credentials and encryption keys, even if the device is physically compromised |
| **TPM 2.0** | Hardware-based cryptographic module | Ensures system integrity and supports secure boot and encryption |

These built-in capabilities establish a multilayer security foundation against evolving threats.

## Advanced sign-in security

Copilot+ PCs support secure, streamlined sign-in options that align with modern identity protection standards:

- **Windows Hello** with biometric authentication  
- **Enhanced Sign-in Security (ESS)** with hardware-backed protection against spoofing and replay attacks  
- **Hardware-backed MDM certificates**  
- **Phishing-resistant authentication**  

These features help reduce the risk of credential theft while enabling fast, secure access.

## Threat prevention by design

Built-in defenses help prevent threats before they can impact operations:

- **App Control for Business**: Helps prevent untrusted or unauthorized applications from running  
- **Protected Print Mode**: Helps secure printing environments by controlling access to sensitive documents  

These hardware protections extend to how AI experiences operate on the device.



As organizations adopt AI across devices and cloud services, they need a consistent approach to management and governance.

Copilot+ PCs, together with Microsoft 365 and Microsoft Intune, enable organizations to apply the same tools, policies, and security frameworks used for devices and applications to AI experiences.

This approach helps ensure that AI-powered capabilities remain secure, compliant, and aligned with organizational requirements as they are introduced and scaled.
