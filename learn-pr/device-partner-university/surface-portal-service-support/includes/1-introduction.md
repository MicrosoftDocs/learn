This module shows you how to create service orders and support requests for Surface devices using the appropriate portal and how to track request status in the correct portal.

## Learning objectives

- Determine when a service order is required and select the correct Surface portal based on device enrollment.
- Submit required service information and distinguish between service and support scenarios.
- Create a support request in the correct portal.
- Identify where service requests are tracked.
- Review service and support status indicators and determine next actions based on status.
