This module shows how to access Surface Support Portal through the Microsoft 365 admin center, verify access requirements, assign required roles, and confirm that the portal is ready to create and manage Surface service and support requests.

>[!NOTE]
> If you're a Microsoft partner, completing this module can also contribute to your organization's skilling visibility when your Microsoft Learn profile is linked to Partner Center.

## Learning objectives

By the end of this module, you'll be able to:

- Access the Surface Support Portal through the Microsoft 365 Admin.
- Verify the access requirements for using Surface Support Portal.
- Assign the administrative roles required to create service orders and support requests.
- Confirm that Surface Support Portal access is ready to use.
