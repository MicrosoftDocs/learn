To use Surface Support Portal, sign in to the Microsoft 365 admin center and confirm that you're working in the correct Microsoft tenant.  A Microsoft tenant is your organization's private space in Microsoft's cloud. It gives your organization access to Microsoft services like:

- Microsoft 365 (Outlook, Teams, SharePoint, etc.)
- Azure Active Directory
- Surface Management and Support Portals

Your organization's tenant is identified by its domain name. It's where users, devices, and data are managed securely. For the Surface Support Portal, access is managed through the Microsoft 365 Admin Center. Let's explore how to access the Microsoft 365 Admin Center:

- Sign in to the Microsoft 365 Admin Center by visiting [https://admin.microsoft.com](https://admin.microsoft.com).
- Confirm that the correct Microsoft tenant is selected.

>[!IMPORTANT]
> The Surface Support Portal on the Microsoft 365 Admin Center doesn't require a subscription for access.

:::image type="content" border="true" source="../media/microsoft-surface-support.png" alt-text="Screenshot of Microsoft Surface Support in the Microsoft 365 Admin Center.":::

- In the left navigation menu, select **Support**.
- Then select **Microsoft Surface support** > **Surface Support Portal**.

>[!IMPORTANT]
> Surface Support Portal is accessed through the Microsoft 365 Admin Center in the Support section.
