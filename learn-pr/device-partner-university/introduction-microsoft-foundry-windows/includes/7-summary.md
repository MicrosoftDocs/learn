Microsoft Foundry on Windows is a unified AI development platform that enables developers to build intelligent applications that run locally on Windows devices while supporting hybrid scenarios with cloud services.

It combines Windows AI APIs, Foundry Local, and Windows ML to simplify AI development across diverse hardware environments and enable efficient AI workloads on CPUs, GPUs, and NPUs.
