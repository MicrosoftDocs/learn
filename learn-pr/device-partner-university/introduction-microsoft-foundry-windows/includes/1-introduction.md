This module introduces Microsoft Foundry on Windows, a platform that helps developers build AI-powered applications that run locally on Windows devices and support hybrid scenarios with cloud services. You learn about the platform’s core components—Windows AI APIs, Foundry Local, and Windows Machine Learning (ML)—and how they work together to simplify AI development across diverse hardware environments.

By the end of this module, you understand when and why to use Microsoft Foundry on Windows and how it supports secure, high-performance AI workloads on Windows devices.

## Learning objectives

By the end of this module, you are able to:

- Identify scenarios where running AI workloads locally or in a hybrid architecture provides advantages for performance, privacy, or resilience.
- Describe the purpose of Microsoft Foundry on Windows as a platform for local and hybrid AI development.
- Explain how Windows AI APIs, Foundry Local, and Windows ML work together to support AI applications on Windows devices.
