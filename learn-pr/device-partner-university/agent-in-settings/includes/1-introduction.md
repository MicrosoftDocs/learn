People often know what they want to accomplish on their PC—such as connecting to a Bluetooth device, making text easier to read, or adjusting display behavior—but they might not know the specific setting that controls it.

Agent in Settings helps streamline that experience on supported Copilot+ PCs by letting users describe what they want to do in natural language. The agent interprets the request, suggests the relevant setting, and—with the user’s approval—can apply the selected change. This makes Settings more intuitive and takes the guesswork out of configuring a device.

In this module, you’ll learn what Agent in Settings is, how it works, and how to use it in everyday scenarios. You’ll also discover when it can be especially helpful for personalization and troubleshooting in Windows 11.

> [!NOTE]
>
> Agent in Settings is currently available only on Windows 11 devices that meet the following criteria:
>
>- The device must be running Windows 11, version 24H2 with [KB5062660](https://support.microsoft.com/topic/9c5bc200-52b6-4c1a-be70-80df6bbfe9c3) or later.
>- The device must be a [Copilot+ PC](https://aka.ms/copilotpluspcs).
>- The device must have the [temporary enterprise feature control](/windows/whats-new/temporary-enterprise-feature-control) policy setting enabled.
>
> The geographic availability of Agent in Settings includes all countries except China.
