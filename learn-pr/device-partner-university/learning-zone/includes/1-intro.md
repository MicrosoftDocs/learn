Microsoft Learning Zone is a new Windows application designed for Copilot+ PCs. It empowers educators to create personalized learning experiences for every student. By combining the power of AI-driven lesson generation with teacher-guided customization, Learning Zone gives educators the ability to design lessons that meet students exactly where they are.

In this module, explore how to download and navigate Microsoft Learning Zone. Discover how to use Learning Zone to generate lessons, manage assignments, use curated content, and analyze student insights. Then learn practical ways to use these features in the classroom to enhance engagement, save time, and personalize learning at scale.

## Learning objectives

By the end of this module, you’ll be able to:

- Create, assign, and manage personalized lessons for students
- Use the Content Gallery and built-in insights to enhance and track learning outcomes
- Describe what Microsoft Learning Zone is and who it’s designed for
