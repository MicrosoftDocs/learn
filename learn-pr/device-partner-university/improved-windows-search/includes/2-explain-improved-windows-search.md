Improved Windows Search delivers a more intelligent and unified search experience across Windows devices. It goes beyond simple keyword matching by surfacing relevant files, settings, and results based on usage patterns, activity, and intent.

This advanced, AI-powered capability is available on Copilot+ PCs, where on-device processing enables more intuitive, natural language search experiences.

> [!NOTE]
> Improved Windows Search is optimized for select languages, including English, Chinese (Simplified), French, German, Japanese, and Spanish.

Traditional search often depends on remembering specific details, such as file names or folder locations. This can slow users down, especially when files are stored across multiple locations or named inconsistently. Improved Windows Search removes that blocker by interpreting intent and context.

For example, instead of searching for an exact file name like:

- Q2_financial_review_v3.pptx

Users can type:

- “Presentation with Q2 numbers”
- “Budget spreadsheet from last week”

Users don’t need to enter full sentences, short phrases or keywords are enough. Windows analyzes the meaning behind the query and returns results based on content, relevance, and recent activity.

Key enhancements include:

- **Faster results** by indexing content more efficiently.

- **More relevant suggestions** based on context and recent activity.

- **Unified search experience** across files, apps, settings, and cloud content, with deeper integration with Microsoft 365 Copilot and OneDrive for Business.

- **Natural language support**, making it easier to find what you need without exact keywords.

These improvements reduce the time users spend navigating folders or switching between applications to locate information.

> [!NOTE]
> Improved Windows Search supports a range of text, image, and document formats, including the file types listed below. This list may expand over time as support continues to evolve.
>
> - Documents: .txt, .pdf, .docx, .doc, .rtf, .pptx, .ppt, .xls, .xlsx
>
> - Images: .jpg/.jpeg, .png, .gif, .bmp, .ico
