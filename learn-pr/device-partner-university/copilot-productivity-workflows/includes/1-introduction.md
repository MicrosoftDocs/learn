Today's business users are using AI tools to increase their productivity more than ever before. These tools help to foster team collaboration, integrate with workplace standards, generate content, and analyze data. This module explores the productivity benefits of Microsoft 365 Copilot in the workplace and how to utilize these features to reduce back and forth.

## Learning objectives

By the end of this module, you'll be able to:

- Explain how Microsoft 365 Copilot supports secure, work-based productivity.
- Use Copilot in Word to summarize documents and identify the next steps.
- Use Copilot in Excel to analyze data and surface insights.
- Use Copilot in Outlook and Teams to summarize communications and support follow-up.
- Use Copilot across apps to create deliverables from meeting outcomes.

>[!NOTE]
> This module contains user interface (UI) elements in Microsoft 365 products. These UI labels may change over time.
