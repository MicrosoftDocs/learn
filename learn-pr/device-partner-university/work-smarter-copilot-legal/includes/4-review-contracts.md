- Identify clauses that might need further review

Use Copilot in Word to review a contract:

1. Open the contract in Microsoft Word.
2. Select the Copilot icon.
3. In the prompt box, enter:
  *“Review this contract and identify clauses that might require further legal review, possible issues or ambiguities, and areas where I might want to consider revisions. Summarize your findings in plain language and cite the relevant contract sections where possible.”*

> [!TIP]

Using this prompt can help create a first-pass summary of potential issues and areas for review. ## Find prior agreements and supporting materials with Improved Windows Search

:::image type="content" source="../media/search.png" alt-text="Screenshot of File Explorer search results showing files related to green bamboo." lightbox="../media/search-expanded.png":::

During your review, you need to find documents that support your work, such as past agreements, regulatory guidance, legal templates, audit findings, or related policies. The problem is you remember what the document is about but not the exact file name.

Improved Windows Search on Copilot+ PCs can help you find files by describing what you’re looking for using natural language. Instead of remembering the exact file name, describe the document the same way you might describe it to a coworker. Windows can interpret your description and help find relevant files.

Finding relevant agreements, policies, and supporting materials more quickly can reduce the time spent gathering information before starting a legal review.

**When to use**: Use Improved Windows Search when you remember the topic, content, or context of a file but not the exact file name.

Do the following to use Improved Windows Search:

1. Press Windows key + S or select the search box on the taskbar.
2. Describe the file you’re looking for.
3. Review the search results.
4. Open the file that best matches what you’re looking for.

> [!TIP]
> After finding the information you need, use Copilot to summarize, analyze, or draft content based on the material you reviewed.

Use searches like these to find relevant materials:
