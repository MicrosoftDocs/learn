If your organization doesn't already have a tenant, the next step is to create one. Do the following to get started:

:::image type="content" border="true" source="../media/surface-support-screen.png" alt-text="Screenshot of Surface Support Portal landing page.":::

- Visit the [onboarding page](https://aka.ms/mstenantcheck) to verify that a tenant doesn't exist.
- Enter your work email address, then select **Next**.
- If your domain isn't recognized, select **Set up account** to create a new tenant.

:::image type="content" border="true" source="../media/tenant-account-creation.png" alt-text="Screenshot of tenant account creation page.":::

- Enter basic organizational details—including first name, last name, business name, and phone number. Once complete, select **Next**.

:::image type="content" border="true" source="../media/create-billing-account.png" alt-text="Screenshot of billing account creation page.":::

- Verify your identity. You can choose to receive a verification code by text or phone call. Select either **Text me** or **Call me**, input your phone number, and then select **Send verification code** to proceed.

>[!NOTE]
> Identity verification is required to confirm that you're authorized to set up the tenant.
