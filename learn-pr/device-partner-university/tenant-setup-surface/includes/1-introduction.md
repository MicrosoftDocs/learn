This module shows how to check the status of a Microsoft tenant and how to create one when needed in the context of managing Surface devices in Surface support portals. You'll learn to evaluate your organization's current tenant configuration, understand when a new tenant is required, and navigate the process of creating one. Additionally, you'll discover how to request and gain access to the Surface Support Portal, an essential tool for device management and support operations.


## Learning objectives

- Determine whether a Microsoft tenant must be created.
- Confirm how to create a new tenant when required.
- Request access to the Surface Support Portal.
