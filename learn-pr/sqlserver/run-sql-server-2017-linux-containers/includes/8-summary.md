In this module, you used a container host to configure and run containers on a Linux host with SQL Server 2019. Containers provide a lightweight and highly scalable way to virtualize multiple services on a single host. 

SQL Server 2017 was the first version of Microsoft SQL Server to support installation on Linux and installation in containers. You now have more hosting choices than ever before for mission-critical databases.
