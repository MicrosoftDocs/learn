Contoso IT staff are migrating Contoso on-premises servers to Windows Server 2019. As a Windows Server administrator, you're responsible for managing and maintaining the server infrastructure that will help Contoso achieve its business goals. As part of this role, you must deploy and maintain DHCP servers to automate the allocation of IP configurations within your organization. 

Having completed this module, you know how to install, configure, and manage the DHCP server role on computers running Windows Server 2019. You also know how to deploy and configure DHCP high availability options.
