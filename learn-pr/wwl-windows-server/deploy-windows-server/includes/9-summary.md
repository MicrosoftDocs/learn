Contoso IT staff are migrating Contoso on-premises servers to Windows Server 2019. As a server administrator, you've been responsible for deploying Windows Server to help Contoso achieve its business goals. After completing this module, you understand the hardware requirements for Windows Server 2019, and you know about the available deployment options. You also understand the difference between Server Core and Server with Desktop Experience, and how you can improve compatibility by installing Features on Demand.
