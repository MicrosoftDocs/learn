Contoso IT staff are migrating Contoso on-premises servers to Windows Server 2019. As part of the migration, you must perform an assessment of the Windows Server editions and recommend an appropriate edition for Contoso. You must also be able to advise Contoso IT on how to support and manage servicing, licensing, and activation after deployment. After completing this module, you can describe the different Windows Server editions, explain servicing channels, and implement licensing and activation.
