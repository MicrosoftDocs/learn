Contoso IT staff are migrating Contoso on-premises servers to Windows Server 2022. As part of the migration, Contoso evaluated its current AD DS environment. As a Windows Server administrator, you've been responsible for managing AD DS objects, such as users, groups, and OUs. You now understand the fundamental AD DS structure and how users, groups, and group managed service accounts relate to OUs.
