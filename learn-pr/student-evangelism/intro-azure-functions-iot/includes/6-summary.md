In this module, our production supervisor originally needed to detect appliances with manufacturing defects visually. However, production needed to be ramped up significantly, and existing manual methods wouldn't work. Further, the company wanted to deploy the solution rapidly and the technology team was well versed in development but not in operations. 

To solve this business problem, cameras upload images to an ingestion service such as Azure Blob Storage, or devices send events through IoT Hub or Event Hubs. Azure Functions runs functions that are triggered by those events to process the workflow in Azure and call an image-analysis workflow suited to manufacturing inspection, such as Azure Content Understanding image analyzers for schema-based image extraction and classification (including manufacturing quality-control scenarios such as detecting defects and anomalies) or a custom image classification or object detection model trained with Azure Machine Learning AutoML. With Flex Consumption, the recommended serverless pay-as-you-go plan for new function apps, Azure Functions can scale event-driven processing without provisioning fixed capacity in advance. The Consumption plan is the legacy serverless hosting option. It's currently the only serverless hosting option for Windows function apps, but [Linux Consumption hosting retires on September 30, 2028](/azure/azure-functions/consumption-plan), so existing Linux Consumption apps should [migrate to Flex Consumption](/azure/azure-functions/migration/migrate-plan-consumption-to-flex). Other hosting plans have different cost and scaling characteristics. Because IoT solutions are event-based, triggers and bindings enabled the company to develop the solution rapidly.

Without Azure Functions processing events from IoT data sources, the company would need to plan more infrastructure in advance and invest more effort in deploying and operating the solution. Using Azure Functions with IoT Hub, Event Hubs, Azure Event Grid, Azure Blob Storage, Azure Content Understanding, and Azure Machine Learning can help the company save costs on suitable pay-as-you-go hosting plans, scale the solution, and deploy rapidly.
