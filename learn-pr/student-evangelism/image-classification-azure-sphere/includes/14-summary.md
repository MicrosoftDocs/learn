In this module, you implemented local whole-image classification on Azure Sphere by using a pre-trained, pre-quantized CIFAR-10 model and one compile-time 32x32 RGB input image. You adapted a real-time capable application to run CMSIS-NN Q7 inference on one of the MT3620 Cortex-M4F real-time cores, built and sideloaded the application for local testing, and viewed the classification result from the device output. You also learned how Azure Sphere supports secured embedded edge devices while keeping this RTApp inference path local to the device.
