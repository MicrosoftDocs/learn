As you are preparing to teach this module, get familiar with what the students will learn during the module. In this module, students will learn how to use the Custom Vision service to create image classification and object detection models. 

This module consists of two lessons: 

* **Lesson 1** – Image classification 

* **Lesson 2** – Object detection
