Running scripts as command jobs and tracking model training with MLflow involves converting notebooks to scripts and experimenting with different logging approaches.

In this exercise, you'll run training scripts and track your work in two parts:

- Run a training script as a command job by converting a notebook to a script, testing it, and running it with parameters.
- Track model training with MLflow by training models with both custom logging and autologging.
