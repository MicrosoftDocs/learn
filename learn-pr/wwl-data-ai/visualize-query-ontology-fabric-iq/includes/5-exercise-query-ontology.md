Now you have an opportunity to practice using the Query builder to query the Lamna Healthcare ontology across its data sources.

In this exercise, you open the relationship graph from the Department entity type, run the default query, and then use the Query builder to answer a specific business question: "Which patients are currently assigned to rooms in the Cardiology department?" You add a Department filter for Cardiology, use the Components pane to include Department, Room, and Patient nodes with their connecting relationship edges, and run the query. You then interpret the results in Diagram, Card, and Table view.

> [!NOTE]
> You need a [Microsoft Fabric trial](/fabric/get-started/fabric-trial) to complete this exercise.

You'll also need to enable the following [tenant settings](/fabric/iq/ontology/overview-tenant-settings):

- Enable Ontology item (preview)
- User can create Graph (preview)

Launch the exercise and follow the instructions.
