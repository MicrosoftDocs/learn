In this exercise, you create a SQL database project, build it locally, push it to a GitHub repository, and configure a GitHub Actions pipeline that automatically builds and deploys schema changes to Azure SQL Database.

> [!NOTE]
>
> To complete this exercise, you need an [Azure subscription](https://azure.microsoft.com/free) and a [GitHub account](https://github.com/join).

Launch the exercise and follow the instructions.
