Now, it's your chance to experience how you can use the CLI (v2) to deploy a model to a managed online endpoint.   

In this exercise, you will:

- Create an online endpoint with the CLI (v2). 
- Deploy the model to the endpoint with the CLI (v2). 
- Test the endpoint with the CLI (v2). 

## Instructions

Follow these instructions to complete the exercise:

1. View the exercise repo at [https://aka.ms/aml-cli2](https://microsoftlearning.github.io/mslearn-aml-cli/?azure-portal=true).
1. Complete the **Deploy an MLflow model to a managed online endpoint** exercise.
