Microsoft Foundry provides two SDKs for building AI applications. The Foundry SDK gives you access to project-level features like agents, evaluations, tracing, and connections. The OpenAI SDK provides model inference with full OpenAI API compatibility.

## Key takeaways

In this module, you learned how to:

- **Use the Foundry SDK** with `AIProjectClient` to access project configuration, connections, tracing, and datasets
- **Use the OpenAI SDK** for model inference, chat completions, and the Responses API
- **Generate responses** with the Responses API and manage conversations
