If you have an Azure subscription, you can use Azure AI Custom Vision to create a custom object detection model for yourself.

> [!NOTE]
> If you don't have an Azure subscription, and you want to explore Azure AI Studio, you can [sign up for an account](https://azure.microsoft.com/free?azure-portal=true), which includes credits for the first 30 days.

Launch the exercise and follow the instructions.

[![Button to launch exercise.](../media/launch-exercise.png)](https://go.microsoft.com/fwlink/?linkid=2296752&azure-portal=true)

When you finish the exercise, don't forget to come back and complete the knowledge check to earn points for completing this module!

> [!TIP]
> After completing the exercise, if you've finished exploring Azure AI services, delete the Azure resources that you created during the exercise.
