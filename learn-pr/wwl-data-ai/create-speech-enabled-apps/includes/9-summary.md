
In this module, you learned how to:

- Connect to Azure Speech in Foundry Tools in a Foundry resource
- Use the Speech to text API to implement speech recognition
- Use the Text to speech API to implement speech synthesis
- Configure audio format and voices
- Use Speech Synthesis Markup Language (SSML)

To learn more about the Azure Speech, refer to the [Azure Speech in Foundry Tools documentation](/azure/ai-services/speech-service/).
