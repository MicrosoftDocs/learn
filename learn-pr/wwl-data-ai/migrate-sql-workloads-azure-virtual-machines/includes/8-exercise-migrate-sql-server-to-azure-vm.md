
Now it's your chance to migrate a SQL Server database using a few of the tools and features available. 

In this exercise, you learn how to migrate a SQL Server database to a SQL Server running on an Azure Virtual Machine.

> [!NOTE]
> To complete this exercise, you'll need an [Azure subscription](https://azure.microsoft.com/pricing/purchase-options/azure-account?cid=msft_learn).

Launch the exercise and follow the instructions.
