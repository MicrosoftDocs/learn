Bot Framework Composer is a visual designer that lets you quickly and easily build sophisticated conversational bots without writing code. The composer is an open-source tool that presents a visual canvas for building bots. It uses the latest SDK features so you can build complex bots with relative ease.

## Learning objectives

In this module, you'll learn how to:

- Understand methods for authoring a bot.
- Plan for a bot dialog flow.
- Use Bot Framework Composer to create a bot.

## Prerequisites

Before starting this module, you should have a general knowledge of bot concepts.
