Now it's your turn to try using tool calling with the Response API and Microsoft Foundry.

In this exercise, you use Microsoft Foundry to deploy a model, and the use the file_search tool to ground its responses in the contents of some files.

> [!NOTE]
> To complete this lab, you need an **[Azure subscription](https://azure.microsoft.com/pricing/purchase-options/azure-account?cid=msft_learn)** in which you have administrative access.
