> This exercise requires an Azure subscription. If you do not already have one, sign up for a free trial at [https://azure.microsoft.com](https://azure.microsoft.com?azure-portal=true).

Now, it's your chance to distribute deep learning in Azure Databricks.

In this exercise, you will:
- Run single-node training with PyTorch and migrate it to Horovod.
- Distribute model training with HorovodRunner.

## Instructions

Follow these instructions to complete the exercise:

1. Open the exercise instructions at [https://learn.microsoft.com/training/modules/deep-learning-azure-databricks/](/training/modules/deep-learning-azure-databricks/).
2. Complete the **Distributed deep learning with Azure Databricks** exercise.
