In this module, you learned how to: 

- Provision an Azure resource for the Azure AI Speech service
- Use the Azure AI Speech to text API to implement speech recognition
- Use the Text to speech API to implement speech synthesis
- Configure audio format and voices
- Use Speech Synthesis Markup Language (SSML)

To learn more about the Azure AI Speech service, refer to the [Azure AI Speech service documentation](/azure/cognitive-services/speech-service/).
