Now you have an opportunity to practice visualizing ontology data in Fabric IQ.

In this exercise, you explore entity instances, visualize the relationship graph, and use the Query builder to filter data and answer a specific business question: "Which patients are currently assigned to rooms in the Cardiology department?" You add filters, select components to control which entity types and relationships appear, and interpret the results in Diagram, Card, and Table view.

> [!NOTE]
> You need a [Microsoft Fabric trial](/fabric/get-started/fabric-trial) to complete this exercise.

You'll also need to enable the following [tenant settings](/fabric/iq/ontology/overview-tenant-settings):

- Enable Ontology item (preview)
- User can create Graph (preview)

Launch the exercise and follow the instructions.
