* Understand the data factory control flow

* Debug data factory pipelines

## Next steps

Data Factory in Microsoft Fabric is the next generation of Azure Data Factory, offering a simpler architecture, built-in AI with Copilot, and deeper integration with the Microsoft Fabric platform. Existing Azure Data Factory pipelines can be migrated to Fabric Data Factory. To learn more, see [What is Data Factory in Microsoft Fabric?](/fabric/data-factory/data-factory-overview), and the [migration planning guide](/fabric/data-factory/migrate-planning-azure-data-factory).
