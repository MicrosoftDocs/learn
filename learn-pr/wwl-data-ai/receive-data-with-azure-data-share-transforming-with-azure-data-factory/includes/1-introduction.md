Learn how to ingest data from Azure Data Share into Azure Data Factory pipelines to build automated ingestion pipelines.

After completing this module, you will be able to:

* Receive data using Azure Data Share
* Ingest data into Azure Data Lake Gen 2 using Azure Data Factory
*	Join and transform data with Mapping Flow in Azure Data Factory
*	Sink a dataset into Azure Synapse Analytics using Azure Data Factory
*	Publish a pipeline run in Azure Data Factory

## Prerequisites

Before taking this module, it is recommended that you complete the following modules:

*	Data Fundamentals
*	Introduction to Azure Data Share
*	Introduction to Azure Data Factory
*	Introduction to Azure Synapse Analytics
