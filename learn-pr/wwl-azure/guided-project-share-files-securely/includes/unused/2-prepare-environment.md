## Prepare environment

Create a resource group and prepare a sample file for upload. Having these ready lets you focus on storage security and SAS token configuration without setup delays.

1.  Sign in to the Azure portal and create resource group rg-gp-file-exchange.
2.  Create a local sample text file named monthly-report.txt for upload.
3.  Confirm your account has permission to create storage resources and generate SAS.
4.  Open one private/incognito browser window for external access testing.

> [!WARNING]
> You're creating billable Azure resources. Use a test subscription and complete the clean-up unit when you're done.

> [!TIP]
> If the portal blade names differ slightly, use the top search bar in the Azure portal.
