## Scenario

Your project team just completed the SAP HANA on Azure (Large Instances) migration kick-off and need to understand the architecture of HANA Large Instances. You present a HANA Large Instances high-level architecture with diagrams that illustrate on how HANA Large Instances provide a non-virtualized, bare metal, high-performance server for the SAP HANA database.

## Learning objectives

You will:

 -  Explore the high-level architecture of SAP HANA on Azure (Large Instances).
 -  Examine the architecture's components.
 -  Examine a sample functional workflow.
