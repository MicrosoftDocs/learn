## Validate success

Review the validation steps you completed during the exercises. Confirm that each item below is true before moving on.

-   [ ] The **gp-rg-readers** security group exists in your tenant.
-   [ ] The new user account is created and you recorded the user principal name (UPN).
-   [ ] The user is a member of **gp-rg-readers**.
-   [ ] The Reader role is assigned to **gp-rg-readers** at the resource group scope.
-   [ ] Check access shows the user's effective permissions at the resource group scope.
-   [ ] Read operations (List, Get) are **Allowed** and create/update/delete operations are **NotAllowed**.
-   [ ] The Activity Log shows a **Create role assignment** entry with your account as the caller.

If any item is not true, return to the relevant exercise and troubleshoot before continuing.
