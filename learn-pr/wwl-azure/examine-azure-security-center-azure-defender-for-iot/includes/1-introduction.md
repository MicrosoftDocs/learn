Azure Defender for IoT integrates with IoT Hub to provide built-in support for the IoT Hub service and enhanced support for devices.

Suppose you have joined a team that is implementing an Azure IoT solution for the company. The company needs your team to enhance the security of the solution. Understanding the Azure services that support security enhancements will help the team to implement the required security provisions. You decide to examine the agent-based implementation of Azure Defender for IoT.

In this module, you will:

 -  Review Azure Security Center concepts.
 -  Examine Azure Defender for IoT options and features of the agent-based implementation.
 -  Examine IoT Hub security alerts and custom alert implementations.

By the end of this module, you will be able to describe the features of an Azure Defender for IoT agent-based implementation.
