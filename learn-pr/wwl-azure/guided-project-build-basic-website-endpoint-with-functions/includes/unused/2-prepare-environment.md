## Prepare environment

Create a resource group and plan your Function App name ahead of time. This preparation ensures you have the namespace reserved and organized before you create the function resources.

1.  Sign in to Azure portal with permission to create Function App resources.
2.  Create a resource group named rg-gp-functions-endpoint.
3.  Decide on a globally unique Function App name, for example func-gp-endpoint-<unique>.
4.  Confirm that you can receive test responses in a browser or API tool.

> [!WARNING]
> You're creating billable Azure resources. Use a test subscription and complete the clean-up unit when you're done.

> [!TIP]
> If the portal blade names differ slightly, use the top search bar in the Azure portal.
