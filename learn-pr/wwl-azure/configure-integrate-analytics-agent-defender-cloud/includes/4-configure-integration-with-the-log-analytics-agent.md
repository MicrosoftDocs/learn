In this lab, learn how to configure and integrate a Log Analytics agent and workspace in Defender for Cloud to collect data from your Azure virtual machines (VMs), Virtual Machine Scale Sets, IaaS containers, and non-Azure (including on-premises) machines to monitor for security vulnerabilities and threats.

> [!NOTE]
> To complete this lab, you will need an [Azure subscription](https://azure.microsoft.com/free)
