The IT operations team at Contoso must secure VM resources both in Azure and on-premises.

In this module, you learned about using the tools available in Azure to assess current Azure resources and OS security configurations. You also learned about enabling Defender for Servers in hybrid environments, onboarding Windows Server computers with Azure Arc, and using Microsoft Defender for Endpoint integration, agentless scanning, and Microsoft Defender Vulnerability Management. Finally, you learned to use Microsoft Defender for Cloud to protect your resources, and about Microsoft Sentinel, SIEM, and SOAR.

Now, you can help the Contoso IT team use Microsoft Defender for Cloud to assess the security configuration of their VM resources and the Windows Server OS that's running on the VMs.
