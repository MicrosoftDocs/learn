## Prepare environment

Create a resource group and test resource as a pilot scope for governance controls. Having your notification email ready lets you fully test budget alerts and policy enforcement.

1.  Sign in with permissions to create budgets and policy assignments.
2.  Create resource group rg-gp-cost-guardrails.
3.  Create a low-cost test resource (for example, a storage account) in the resource group.
4.  Confirm email address to receive budget alert notifications.

> [!WARNING]
> You're creating billable Azure resources. Use a test subscription and complete the clean-up unit when you're done.

> [!TIP]
> If the portal blade names differ slightly, use the top search bar in the Azure portal.
