In this module, you learned how to generate code using GitHub Copilot Chat for Visual Studio Code. You used GitHub Copilot to generate ghost text suggestions and next edit suggestions (NES) as you wrote and edited code, and to generate code suggestions using chat interactions. You learned how to accept, partially accept, and discard suggestions. You also explored the various chat features of GitHub Copilot, including the Chat view's Ask, Plan, and Agent modes, as well as inline chat, Quick Chat, and smart actions.

The main takeaways from this module are the benefits and features of GitHub Copilot. You learned how to use GitHub Copilot to write code more quickly and accurately. You learned how to accept, partially accept, and discard suggestions. You also gained practical experience using Visual Studio Code and GitHub Copilot to develop C# applications.
