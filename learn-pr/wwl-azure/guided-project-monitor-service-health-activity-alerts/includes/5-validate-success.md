-   [ ] The test notification status shows **Success** and you received the email notification.
-   [ ] You reviewed all four Service Health tabs (Service issues, Planned maintenance, Health advisories, Health history).

-   [ ] The alert condition for the Activity Log alert detects **Delete resource group** events.

If any item is not true, return to the relevant exercise and troubleshoot before continuing.
