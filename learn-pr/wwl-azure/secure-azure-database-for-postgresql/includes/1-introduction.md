Azure Database for PostgreSQL uses multiple layers of security to protect data. In this module you will learn about the Azure Database for PostgreSQL security module, the built-in server roles, and how to grant permissions to database users. You will also learn about how encryption protects your data at rest, and in transit. 

In this module, you will learn how to:

- Describe Azure Database for PostgreSQL security.
- Describe built-in Azure Database for PostgreSQL server roles.
- Grant permissions in Azure Database for PostgreSQL.
- Understand encryption in Azure Database for PostgreSQL.
- Configure Azure Active Directory authentication.
