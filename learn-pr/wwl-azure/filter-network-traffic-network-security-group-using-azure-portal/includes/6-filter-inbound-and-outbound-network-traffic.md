Now it's time to apply what you learned about network security groups. In this exercise, you create a virtual network infrastructure, configure network security groups with inbound and outbound rules, and test how traffic filtering works in practice.

## Exercise objectives

In this exercise, you'll:

- Create and configure network security groups
- Define security rules to allow and deny specific traffic
- Associate NSGs with subnets and network interfaces
- Test and verify traffic filtering behavior
